  1. Genera una variable codigoSecreto de tipo array de 5 número aleatorios entre 1 y 10 usando la libreria Math.random();

  2. Programa la función que te indique cuantas veces sale un número concreto (se lo podemos pedir al usuario) dentro del array.

  3. Pide al usuario una posición de 1 a 5 y un número de 1 a 10. Crea una función que indique si ese número está o no en la posición indicada del array.

  4. Modifica el punto anterior y para cada posición del array pide al usuario un número y guardarlo en un array de 5. Crea un algoritmo para que nos indique para cada posición cuantas veces sale ese número, pero si está ahí simplemente indique “Ok”.
